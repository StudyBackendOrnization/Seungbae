# 마방진

## 기능

- 고정된 크기를 사용한다
- 숫자를 입력으로 받는다. 최대 9까지

## 조건

- 정수 입력만 허용한다.
- 숫자가 아닌 문자,실수를 입력할시 다시 시도하세요. 문자 출력
